# Système de Gestion de Stock

```text
📦 PROJET : GESTION DE STOCK
 ┣ 🎯 OBJECTIF
 ┃ ┗ Simplifier la gestion des stocks, entrées/sorties et accès utilisateurs
 ┃
 ┣ 💻 TECHNOLOGIES
 ┃ ┣ Backend  : Laravel (PHP)
 ┃ ┣ Frontend : HTML / CSS / JS (Template prêt à l'emploi)
 ┃ ┗ Vues     : Blade
 ┃
 ┣ ✨ FONCTIONNALITÉS
 ┃ ┣ 🔐 Authentification (Connexion/Inscription)
 ┃ ┣ 📁 Catégories (Créer, Modifier, Supprimer)
 ┃ ┣ 🏷️ Produits (Ajouter, Editer, Informations)
 ┃ ┗ 📊 Stock (Quantités, Alertes, Historique)
 ┃
 ┗ 🚀 INSTALLATION & DÉMARRAGE
   ┣ 1️⃣ Cloner / Télécharger le projet
   ┣ 2️⃣ Installer dépendances : `composer install`
   ┣ 3️⃣ Configurer : Copier `.env.example` en `.env` + Clé `php artisan key:generate`
   ┣ 4️⃣ Base de données  : `php artisan migrate`
   ┗ 5️⃣ Lancer serveur   : `php artisan serve`
```
